# Axe API

This is a simple Axe API project. It contains only CRUD actions. This project has been created for a blog post; [How to create a simple CRUD application in AxeÂ API?](https://medium.com/p/6164ca0cc393)

## Usage

```
$ npm install
$ npm install -g knex nodemon
$ knex --esm migrate:latest
$ npm run start:dev
```

Please visit the [http://localhost:3000/api/users](http://localhost:3000/api/users).
